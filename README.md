## Getting Started

To get started with this project, follow these steps:

1. **Prerequisites**:
   - Node.js
   - npm or Yarn

2. **Installation**:
   - Clone the repository:

     ```sh
     git clone https://github.com/AbdulAzeez002/BookStore.git
     ```

   - Navigate to the project directory:

     ```sh
     cd BookStore
     ```

   - Install project dependencies:

     ```sh
     npm install
     # OR
     yarn install
     ```
